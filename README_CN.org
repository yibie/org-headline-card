* Org Headline Card

将 Org-mode 标题及其内容转换为精美的预览卡片图片。

** v0.3 更新要点

- 渲染引擎：PlantUML → SVG + ImageMagick（不再依赖 Java/PlantUML）
- 导出速度：相同内容长度从约 1~3s 降到 1s 以内（典型值）
- 图片体积：相同内容长度从约 1MB 降到约 10~40KB（典型值）
- 新增交互式预览与可导出的特效（高亮、光晕、描边框、波浪线）

** 功能特点

- 将 Org 标题转换为可视化卡片
- 支持 Org 标记语法（粗体、斜体、链接等）
- 多种预设主题风格
  - 现代中文风格 (chinese-modern)
  - 水墨风格 (chinese-ink)
  - 屏风风格 (chinese-screen)
- 明亮风格 (light)
- 暗黑风格 (dark)
- 暖色风格 (warm)
- 固定宽度导出（卡片宽度一致）
- 支持自定义主题
- 将链接从正文中提取，以灰色字体列表形式显示在卡片底部

** 基本使用

1. 将光标移动到任意 Org 标题
2. 执行 =M-x org-headline-card-at-point=
3. 生成的卡片图片将保存在 =~/.emacs.d/headline-cards/= 目录下

*** 预览（Preview）

1. 将光标移动到任意 Org 标题
2. 执行 =M-x org-headline-card-preview=
3. 在预览缓冲区中可用快捷键：
   - =q= 退出
   - =r= 刷新
   - =t= 切换主题
   - =b= 切换边框
   - =e= 编辑内容
   - =RET= 导出图片
   - =h= 背景高亮（特效）
   - =g= 文字光晕（特效）
   - =s= 文本描边框（特效）
   - =w= 波浪下划线（特效）
   - =0= 移除光标处的特效
   - =C-c C-c= 清空所有特效

*** 添加更多特效（预览中）

- 选中区域（或将光标放在单词上）后按 =h/g/s/w= 应用特效
- 特效会随 =RET= 导出到图片中
- 特效仅支持单行（跨行选择会报错）
- 通过前缀参数自定义颜色/宽度：例如 =C-u h=、=C-u s=

** 主题

*** 内置主题

提供六种精心设计的主题风格：

- chinese-modern :: 现代中文风格，适合一般文档展示
- chinese-ink :: 水墨风格，适合诗词、散文等文学内容
- chinese-screen :: 屏风风格，适合古典文献展示
- light :: 明亮风格，适合日常使用
- dark :: 暗黑风格，适合夜间模式
- warm :: 暖色风格，适合长时间阅读

切换主题：

使用 `M-x org-headline-card-set-theme` 切换主题

也可以将常用主题添加到 emacs 配置文件中，如： 
#+begin_src emacs-lisp
(org-headline-card-set-theme 'chinese-ink)
#+end_src

*** 自定义主题

**** 调整基础参数
#+begin_src emacs-lisp
(setq org-headline-card-base-theme
      '((padding . "40")                  ; 内边距
        (roundCorner . "40")              ; 圆角
        (titleFontSize . "32")            ; 标题字号
        (contentFontSize . "26")))        ; 正文字号
#+end_src

**** 创建个性化主题
#+begin_src emacs-lisp
;; 添加一个浅紫色主题
(add-to-list 'org-headline-card-themes
             '(purple-light . ((defaultFontName . "LXGW WenKai Mono")
                               (defaultFontSize . "16")
                               (backgroundColor . "#F8F5FF")
                               (rectangleBorderColor . "#E6E0F3")
                               (rectangleFontColor . "#2C2C2C")
                               (rectangleBackgroundColor . "#FDFAFF"))))
#+end_src

主题可用参数：
- =defaultFontName= :: 字体名称
- =defaultFontSize= :: 默认字号
- =backgroundColor= :: 背景色
- =rectangleBorderColor= :: 边框颜色
- =rectangleFontColor= :: 文字颜色
- =rectangleBackgroundColor= :: 卡片背景色
- =backgroundGradient= :: 背景渐变（SVG 格式）
- =backgroundGradientType= :: 渐变类型（'linear 或 'radial）
- =backgroundImage= :: 背景图片路径
- =backgroundImageOpacity= :: 背景图片透明度
- =backgroundPattern= :: 内置纹理（'dots, 'grid, 'lines, 'cross-lines, 'waves）
- =backgroundPatternColor= :: 纹理颜色
- =backgroundPatternOpacity= :: 纹理透明度

** 导出配置

#+begin_src emacs-lisp
;; 导出图片最大宽度（字符数，默认 50）
(setq org-headline-card-export-width 50)

;; 导出图片 DPI（默认 72，可设为 144 获得 2x 清晰度）
(setq org-headline-card-export-dpi 72)

;; 导出字号（默认 16）
(setq org-headline-card-export-font-size 16)
#+end_src

** 依赖

- Emacs 27.1+
- Org-mode 9.4+
- ImageMagick（convert 或 magick）

安装 ImageMagick：
- macOS: =brew install imagemagick=
- Ubuntu/Debian: =sudo apt-get install imagemagick=
- Arch Linux: =sudo pacman -S imagemagick=
- Windows: 从 https://imagemagick.org/script/download.php 下载

小提示：可以运行 =M-x org-headline-card-check-dependencies= 检查依赖是否齐全。
